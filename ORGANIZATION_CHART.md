## Communication Flow

### Upward
- Sub-agents report to their Head agent
- Heads report to MAX
- MAX reports to Amadu

### Horizontal
- Sub-agents can collaborate across departments (NOVA-COMPETITOR talks to PULSE-ADS)
- Heads coordinate with each other (ARIA + QUINN collaborate)
- MAX facilitates cross-team work

### Downward
- MAX briefs Heads with strategic context
- Heads brief their specialists with specific tasks
- Specialists execute and report results back up

---

## How Tasks Flow

**Example: "Build a website for a plumbing company"**

1. **Amadu → MAX:** "Build plumbing website"
2. **MAX** thinks strategically:
   - Need design (ARIA team)
   - Need copy (QUINN team)
   - Maybe need research (NOVA team)
   - Need humanization (ECHO team)
3. **MAX → ARIA:** "Design plumbing website. Local service feel. Trust-building visuals."
4. **ARIA → ARIA-WEB, ARIA-BRAND, ARIA-LAYOUT:** Each specialist owns their piece
5. **MAX → QUINN:** "Write website copy. Show credibility + urgency."
6. **QUINN → QUINN-COPY, QUINN-CONTENT:** Copy team executes
7. **MAX → ECHO:** "Polish everything so it feels local & human."
8. **ECHO → ECHO-TONE, ECHO-FLOW, ECHO-AUTHENTIC:** Polish team refines
9. **Results flow back up** → ARIA gives design → QUINN gives copy → ECHO polishes
10. **MAX synthesizes** → presents to Amadu
11. **Amadu reviews** → feedback → MAX learns & adapts for next project
