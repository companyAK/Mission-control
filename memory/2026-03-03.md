## Lessons From Today (CRITICAL)

**Problems that happened:**
1. SQLite (dev) vs PostgreSQL (prod) → Build succeeded, 404 in production
2. Duplicate NextAuth handlers → Pages router + App router conflict
3. Type errors from schema mismatches

**Root causes:**
- No testing on final infrastructure
- No QA layer
- Assumptions about environments

**Prevention (now in place):**
- BYTE-QA tests on BOTH dev AND prod before code is presented
- Pre-Flight checklist prevents misconfiguration
- Master-QA tracks patterns and updates procedures
- Architecture approved by Amadu before any coding starts
